Data Acquisition

Land Use Map

A land use geojson file was obtained from Lincoln’s open data site (link).

This data does not appear to have parcel numbers, so it is not as useful as anticipated.

Contextual Maps

A map of Nebraska roads was obtained from Nebraska’s open data site to provide contextual cues for generated maps. Smaller roads were added from Lancaster county’s street data.

A map of Nebraska rivers was obtained from the state open data site for additional contextual information. Lakes and other static bodies of water were acquired using waterbodies data

A map of nebraska counties was obtained and used to crop the roads and rivers information to Lancaster county in order to save space and allow for upload of the dependent data files to GitHub. Municipal boundaries were sourced from state records and added for additional context.

Property Assessment Records

Two xlsx files were acquired from Doug R. Guess of the Lancaster County GIS office on 2024-03-13; the total cost of the database search was $60 (+ $2 processing fee). The information is publicly available at https://app.lincoln.ne.gov/aspx/cnty/cto/, but assembling the full dataset would require assembling data from approximately 1.7576^{4} queries; it was deemed reasonable to pay for the data rather than assemble it manually.

  • data/SusanVanderplas.xlsx contains 126479 rows and 126479 columns, including the following columns not present in the initial data file: PrimeUseDesc, PrimeUse, Situs, Prop_ClassDesc, Total_Acres
  • data/SusanVanderplas-New.xlsx contains 128511 rows and 128511 columns, including the following columns not present in the initial data file: PrimaryUseDesc, SitusAddress, PropertyClassDesc, TotalAcres, ComBldgNo, ComBldgTypeDesc, ComNoUnits

Further analysis shows that

  • PrimeUseDesc and PrimaryUseDesc have the same 28 levels (and are thus most likely duplicated columns)
  • Prop_ClassDesc and PropertyClassDesc have the same 8 levels

Joining the two datasets on common variables allowed verification that (TotalAcres, Total_Acres) and (SitusAddress, Situs) contain the same information. In addition, joining the two datasets on all variables did not produce any non-matching rows, indicating that data/SusanVanderPlas.xlsx contains a subset of the data present in data/SusanVanderPlas-New.xlsx. What is curious is that data/SusanVanderPlas-New.xlsx contains more rows and columns than data/SusanVanderplas.xlsx but takes up approximately 0.3 MB less space on disk.

After joining the two data tables to ensure that the maximum information available is analyzed, there are 128511 records with 39 available variables.

Breaking this summary down, we can see associations between variable missingness, which likely occurs due to the difference between residential and commercial properties, assessments, and valuation. Examining this hypothesis with a 10% sample of the data, clustering observations by missingness, we can see there is a group of observations that is largely missing the assessment information for residential housing, such as ResTypeDesc, …, GarageCap, but these observations are more likely to have ComBldgNo, …, ComNoUnits; there is also a group which seems to lack both sets of information.

Lincoln/Lancaster County Parcel GIS

Parcel-level GIS maps are available at https://maps.lincoln.ne.gov/default/index.html?viewer=GISViewer. These data were systematically downloaded using the draw rectangle tool, and the zip files are in data/shapefiles/*.

It may make sense to systematically download any missing files via script, as spot-checks of missing parcels suggest the information is present in the system and may just not be exporting properly. Of the 17460 missing parcels in the data, 15887 are residential and may potentially affect the analysis. Figure 2 shows the property class type of properties with available spatial data.
